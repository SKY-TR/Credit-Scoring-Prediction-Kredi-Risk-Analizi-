# Bank Credit Scoring Prediction Project (Kredi Risk Analizi)

This project is a machine learning implementation focused on financial risk assessment. As a Mathematics and Computer Science student, I developed this model to simulate how banks evaluate loan applicants.

##  Project Overview
In the banking industry, predicting the likelihood of a loan default is crucial for maintaining financial stability. This project uses a **Random Forest Classifier** to analyze customer data and predict credit risk.

##  Technical Features
- **Data Simulation:** Created a synthetic dataset reflecting real-world banking variables (Income, Credit Score, Loan Amount).
- **Algorithm:** Utilized `Random Forest` for its high accuracy in classification tasks.
- **Evaluation:** Focused on precision and recall to ensure reliable risk detection.

##  Tech Stack
- **Language:** Python
- **Libraries:** Pandas, Scikit-learn, NumPy
